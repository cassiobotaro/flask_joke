# :joy: Flask Joke
The Funniest Joke in the Flask World

Based on [python packaging](https://python-packaging.readthedocs.io).


## Usage

```python
>>> import flask_joke
>>> print(flask_joke.joke())
>>> print(flask_joke.joke_from_json())
```
